# DGColor Business System

## Project Overview
The DGColor Business System is a web-based application developed to support the daily operations of a photography and printing studio. The system helps streamline processes such as studio bookings, printing services, and customer management.

This system replaces manual record-keeping with a centralized digital platform, improving efficiency, accuracy, and overall service quality.

---

## Objectives
- To digitalize studio booking and printing service management
- To improve workflow efficiency and reduce manual errors
- To provide a centralized dashboard for better monitoring
- To enhance customer service through organized data management

---

## ⚙️ Features

### Authentication
- Secure login system for Admin and Staff
- Role-based access control

### Dashboard
- Overview of bookings and printing orders
- Quick access to system modules

### Studio Management
- Manage studio packages
- Handle customer bookings
- Generate booking records

### Printing Services
- Manage print orders
- Track order details and status

### Customer Management
- Register and manage customer information

## User Roles

### Admin
- Manage studio packages
- Manage printing services
- View all reports
- Full system access

### Staff
- Manage studio bookings
- Manage printing orders
- View dashboard

## How to Run the System

1. Clone the repository:
   
2. Open the project in Visual Studio

3. Configure the database:
- Open `appsettings.json`
- Update your SQL Server connection string

4. Run database:
- Import `DGColor.sql` into SQL Server

5. Run the project:
- Press `Ctrl + F5` or click **Start**

---

## Limitations
- System runs on localhost environment
- No online payment integration
- Limited to internal staff usage
